| **Vị trí**                               | **Loại vấn đề**                      | **Mô tả hậu quả**                                                                                                                           |
|------------------------------------------|--------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| \<%! private int requestCounter = 0; %\> | **Race Condition / Shared State**    | Biến này là **instance variable của Servlet**, dùng chung cho tất cả request đa luồng sẽ gây sai số, không thread-safe, có thể loạn dữ liệu |
| \<% requestCounter++; %\>                | **Business logic trong View**        | View không được phép xử lý logic vi phạm MVC, khó test, khó maintain                                                                        |
| \<% List\<Student\> list = \... %\>      | **Scriptlet (Java code trong JSP)**  | Trộn Java vào HTML code rối, không tách biệt FE/BE, khó đọc                                                                                 |
| \<%= sv.getFullName() %\>                | **XSS (Cross-Site Scripting)**       | Nếu dữ liệu chứa \<script\> sẽ render trực tiếp lỗ hổng bảo mật nghiêm trọng                                                                |
| \<%= sv.getScore()%\>;                   | **Syntax bug (HTML output sai)**     | Dư dấu ; hiển thị sai UI (85;)                                                                                                              |
| for (\...) { if (\...) { \... } }        | **Business logic trong View (nặng)** | Xử lý xếp loại ngay JSP vi phạm "Thin View", đáng lẽ phải ở Controller/Service                                                              |
| \<!\-- Tiêu đề trang báo cáo \--\>       | **Sai loại comment**                 | HTML comment vẫn render ra client lộ thông tin nội bộ                                                                                       |
| \<%= title %\>                           | **Expression thay vì EL**            | Không tận dụng EL khó đọc, không tận dụng JSTL ecosystem                                                                                    |
